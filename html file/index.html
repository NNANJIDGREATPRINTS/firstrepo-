INTRODUCTION

HTML is known as HyperText Markup Language. HTML's primary function is to define the structure of a document using a tagging system; allows web browsers to present web pages using a standardised language. which is Originally used to create content and apply style information to web pages: defines which parts of a document are headings, paragraphs, tables etc; and can be used to define how each section is presented.

Latest iteration is HTML5, which introduced tags for multimedia content to be displayed natively in HTML.

HTML TAGS

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content. HTML tags contain three main parts:

  • Opening tag,
  • Content
  • And closing tag.

NOTE: some HTML tags are unclosed tags.

When a web browser reads an HTML document, browser reads it from top to bottom and left to right. HTML tags are used to create HTML documents and render their properties. Each HTML tags have different properties.
An HTML file must have some essential tags so that web browser can differentiate between a simple text and HTML text. You can use as many tags you want as per your code requirement.
